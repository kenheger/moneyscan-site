AI Literacy Assessment (Full Interactive Version)
Take the Assessment Answer each question honestly. Scoring: 0 = Not at all / Never 1 = Somewhat / Occasionally 2 = Comfortable / Regularly 3 = Very confident / Frequently
Questions:
1.	How comfortable are you using AI tools like ChatGPT, Grok, or Claude for everyday tasks (writing, research, brainstorming)?
2.	Can you write an effective, detailed prompt to get useful and accurate output from an AI model?
3.	How familiar are you with basic AI concepts such as machine learning, generative AI, and training data?
4.	Have you used AI to analyze data, create summaries, or automate repetitive work?
5.	How confident are you spotting potential biases, inaccuracies, or hallucinations in AI-generated content?
6.	Do you understand the difference between using AI as a helpful tool versus letting it make important decisions for you?
7.	Have you experimented with AI tools for image, video, or audio generation?
8.	How quickly can you learn and start using a brand-new AI tool when it is released?
9.	How familiar are you with basic privacy, copyright, and ethical considerations when using AI?
10.	Have you used AI tools to help with career-related tasks (resume writing, job search, interview prep)?
11.	How confident are you combining AI tools with your current professional expertise or domain knowledge?
12.	Are you actively experimenting with AI to create side income or improve your work output?
